# 🎓 Cadastro de Alunos – Sistema Web em Ruby

Aplicação web desenvolvida em Ruby para gerenciamento de alunos, permitindo cadastro, visualização e manipulação de registros por meio de uma interface simples e funcional.

O sistema foi construído com foco em organização de dados e prática de desenvolvimento web, utilizando templates dinâmicos e estrutura baseada em separação de responsabilidades.

---

## 🚀 Funcionalidades

* Cadastro de alunos
* Listagem de registros
* Visualização de informações
* Remoção de dados
* Interface web com interação dinâmica
* Sistema de navegação com controle de sessão (logout funcional)

---

## 🛠️ Tecnologias Utilizadas

* Ruby
* ERB (Embedded Ruby Templates)
* HTML, CSS e JavaScript
* Estrutura web com arquivos públicos (assets)
* Bundler (Gerenciamento de dependências)

---

## 📂 Estrutura do Projeto

```bash
cadastro-de-alunos/
│
├── public/          # Arquivos estáticos (CSS, JS, imagens)
│   ├── imgs/
│   ├── style.css
│   └── script_js
│
├── views/           # Templates da aplicação (ERB)
│   └── index.erb
│
├── app.rb           # Arquivo principal da aplicação
├── Gemfile          # Dependências do projeto
└── Gemfile.lock
```

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/CarlosESoares/Cadastro-de-Alunos
```

2. Acesse a pasta do projeto:

```bash
cd Cadastro-de-Alunos
```

3. Instale as dependências:

```bash
bundle install
```

4. Execute a aplicação:

```bash
ruby app.rb
```

5. Acesse no navegador:

```bash
http://localhost:4567
```

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido para consolidar conhecimentos em desenvolvimento web com Ruby, abordando:

* Estruturação de aplicações web
* Separação entre lógica e interface (views)
* Manipulação de dados
* Interação com o usuário via interface web
